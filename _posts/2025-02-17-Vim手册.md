## LearnDocs 🇨🇳🇨🇳🇨🇳

- 启动配置

```shell
# 使用默认的配置启动vim
vim -u NONE -N

# 使用指定的配置文件启动vim
vim -u ~/essential.vim
```

> vimrc

```bash
# essential.vim内容
set nocompatible
filetype plugin on

# 所有的数字都识别为十进制，不管是不是0开头的
set nrformats=
```

- 查看版本信息

```shell
:version
```
